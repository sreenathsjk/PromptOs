# PromptOS — AI Product Architect Platform

> **Stop prompting. Start building.**

PromptOS is an intelligent AI Product Architect platform that deeply understands your idea through adaptive questioning and generates ultra-optimized, platform-specific prompts for building websites, apps, SaaS products, AI tools, and more.

---

## 🚀 What is PromptOS?

Most people struggle to explain what they want to AI coding tools. PromptOS solves this by acting like a senior product team before generating any prompt — asking the right questions, structuring your vision, and then producing professional-grade prompts tailored to whichever AI platform you're building with.

Think of it as **ChatGPT meets Typeform meets a Silicon Valley product consultant** — focused entirely on intelligent context extraction and prompt synthesis.

## 🎯 Core Features

### Adaptive Question Engine
- Detects your project type (school, food delivery, CRM, SaaS, AI tool, etc.)
- Asks only relevant, targeted questions
- Supports tap-to-select chips for platform, style, and Yes/No answers
- Feels like talking to an elite product strategist

### 7 Specialized AI Agents
| Agent | Role |
|---|---|
| 🏗 Product Architect | App structure, workflows, feature planning |
| 🎨 UI/UX Expert | Design systems, animations, layouts |
| ⌨ Senior Developer | Architecture, tech stack, APIs, databases |
| 🚀 Startup Advisor | Monetization, growth systems, virality |
| 📈 SEO & Marketing | SEO structure, landing pages, conversion |
| ✦ Prompt Engineer | Final prompt optimization, context enhancement |
| ☁ Deployment Agent | Hosting, CI/CD, cloud architecture |

### 8 Prompt Types Generated Per Project
1. **Master Prompt** — Full project context and vision
2. **Frontend** — UI components, design system, animations
3. **Backend** — API routes, auth, middleware, security
4. **Database** — PostgreSQL schema, indexes, RLS policies
5. **UI/UX** — Design tokens, motion specs, breakpoints
6. **Branding** — Identity, voice, copy guidelines, social proof
7. **Deployment** — Vercel, Supabase, Cloudflare, CI/CD pipeline
8. **AI Agent** — Model selection, embeddings, vector DB, safety

### Platform-Specific Optimization
Prompts are intelligently tuned for each target platform:

| Platform | Optimization Focus |
|---|---|
| 💜 Lovable | Visual UI descriptions, color systems, UX flow |
| ⚡ Bolt | Feature structure, app flow logic, implementation |
| ◎ Cursor | Modular engineering tasks, clean architecture |
| ▲ v0 | React components, Tailwind classes, UI hierarchy |
| 🔁 Replit Agent | Deployable backend, environment setup, infra |
| ✦ Claude | Reasoning chains, detailed context, step-by-step |
| ◉ GPT-4o | Role context, output format, concrete examples |
| ◈ Gemini | JSON schemas, multimodal context, concise |

## ⚡ Getting Started

### Option 1 — Open directly
Download `PromptOS.html` and open it in any browser on mobile or desktop.

### Option 2 — Host on GitHub Pages
1. Push `PromptOS.html` to a GitHub repo
2. Go to **Settings → Pages → Deploy from branch**
3. Select `main` branch, root folder
4. Your app is live at `https://username.github.io/repo-name/PromptOS.html`

### Option 3 — Deploy to Vercel
1. Push to GitHub
2. Import repo in [vercel.com](https://vercel.com)
3. Set output to root, framework to "Other"
4. Done — live in 30 seconds
